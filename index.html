Informations Pratiques

Quelques détails pour vous aider à préparer votre venue.

Registre de Mariage

Votre présence à nos côtés est déjà le plus beau des cadeaux ! Si toutefois vous souhaitez exprimer votre affection par un geste, une boîte de contribution (enveloppes physiques) sera installée lors de la réception.

Code Vestimentaire

Aucun code vestimentaire ou couleur spécifique n'est imposé pour la journée.

Nous recommandons simplement à nos chers invités d'opter pour une tenue élégante, chic et digne d'une telle célébration. Les tenues décontractées (t-shirts, shorts, jeans, baskets de sport) sont à éviter pour le bon déroulement de l'événement.

Retransmission Virtuelle
